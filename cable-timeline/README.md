# Computer Cable and Connector Timeline

An educational infographic illustrating the historical development of major computer cables and connectors, showing when technologies were introduced, widely adopted, and replaced by newer standards.

## Purpose

This project provides a visual timeline of computer connectivity technologies, helping learners understand how cables and connectors evolved over time to support increasing demands for speed, functionality, and convenience.

## Technologies Included

- Serial Ports (RS-232)
- Parallel Ports
- VGA
- DVI
- HDMI
- Ethernet
- USB Standards
- USB-C
- Thunderbolt
- Fiber Optic Connectors

## Learning Objectives

After reviewing this infographic, learners should be able to:

- Understand the chronological evolution of computer connectivity
- Identify major milestones in cable and connector development
- Compare legacy and modern connection standards
- Recognize how technological advancements improved performance and functionality
