# Computer Motherboard Components

An educational presentation designed to introduce beginners to the major components found on a computer motherboard and explain their purpose within a modern computer system.

## Purpose

This project was created to help students and aspiring IT professionals identify motherboard components and understand how they interact to support system operation.

## Topics Covered

- CPU Socket
- RAM (DIMM) Slots
- Chipset
- PCIe Expansion Slots
- M.2 Slots
- SATA Connectors
- Power Connectors
- BIOS/UEFI
- CMOS Battery
- Front Panel Headers
- USB Headers
- Fan Headers
- Rear I/O Ports

## Learning Objectives

After reviewing this presentation, learners should be able to:

- Identify major motherboard components
- Understand the purpose of each component
- Recognize common motherboard layouts
- Explain how components communicate within a computer system
