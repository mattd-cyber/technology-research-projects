# History of Computer Cables and Connectors

This project explores the development of computer cables and connectors from early computing technologies through modern high-speed standards.

Topics covered include:

- Serial Ports (RS-232)
- Parallel Ports
- VGA
- DVI
- HDMI
- USB Standards
- USB-C
- Thunderbolt
- Ethernet
- Fiber Optic Connectors

The presentation analyzes the purpose, adoption timeline, technological advancements, and impact of each technology on computer connectivity.
