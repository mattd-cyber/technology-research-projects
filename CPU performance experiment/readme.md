# CPU Performance Experiment

## Overview

This project investigates how different applications utilize CPU resources and affect overall system performance. Using built-in performance monitoring tools, CPU activity was observed and analyzed under various workloads to better understand processor behavior and resource management.

## Topics Covered

* CPU Utilization
* Multi-Core Processing
* Multithreading
* Performance Monitoring
* System Resource Management
* Troubleshooting Performance Issues

## Key Takeaway

Monitoring CPU performance helps identify resource-intensive applications, understand system behavior under load, and diagnose performance bottlenecks that impact responsiveness and stability.
