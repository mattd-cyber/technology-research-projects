# Computer Cable Visual Guide

An educational presentation designed to help beginners identify common computer cables and connectors used in modern and legacy computer systems.

## Purpose

This project provides a visual reference guide to computer cables and connectors, explaining their appearance, purpose, and typical use cases.

## Topics Covered

- USB Type-A
- USB Type-B
- USB-C
- Micro USB
- Mini USB
- HDMI
- DisplayPort
- VGA
- DVI
- Ethernet (RJ-45)
- SATA
- Audio Connectors
- Power Connectors
- Fiber Optic Connectors

## Learning Objectives

After reviewing this presentation, learners should be able to:

- Identify common computer cables and connectors
- Understand the purpose of each cable type
- Recognize modern and legacy connectivity standards
- Select the appropriate cable for common computing tasks
